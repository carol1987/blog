maxlength

maxlength 属性, 指定用户可输入最大长度.

size

size 属性, 指定界面上控件的显示的长度. 默认值为 20.

readonly

readonley 属性, 表示控件处于只读状态, 不可编辑; 会正常提交到服务器;

list

list 属性, 属性值为 datalist 标签的 id 值; datalist 表示含有 list 属性的控件的预设值; 同时 pattern 会影响 list 提示(用户预设值不合法时);

placeholder and title

placeholder 属性, 指定值为以文本形式显示在控件中, 起提示作用;

label(placeholder不能代替label的作用), placeholder(短文本, 起提示作用), title(更详细的描述)

pattern

自定义约束规则. 用来扩展 required 等内置规则; 支持 JavaScript 语言中正则表达式定义.

min, max, step

定义控件最小值, 最大值和步值; value 合法取值范围为: min + step * n(n为整数); 不存在 min 时, value 约束计算时, min 取 0.

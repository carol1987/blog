JavaScript 编码

JavaScript 文件中的编码

  • 支持 unicode 表示 - \u0065. [ECMAScript]
  • 支持以 \x 开头的十六进制表示.
  • 支持以 \ 开头的 8 进制表示.
  • 支持 c 语言编码: \b 表示退格. \n ...

HTML 中的 inline-script 编码

先使用 script 编码方式编码, 再使用 HTML 编码方式编码
需要注意使用 server 端语言生成到 <script>中的 JavaScript 代码.
不推荐在 html 标签中绑定 script 代码(事件/伪协议等)

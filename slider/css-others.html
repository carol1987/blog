SASS

说在开头

  • CSS 可以解决问题时, 不必要使用预处理工具.
  • 当在使用 CSS 的很复杂, 不再容易开发或维护时, 可使用工具来辅助完成.
  • 如何把预处理工具整合到项目工程中, 就是另外的问题.

SASS: CSS 预处理器, 即在上线部署之前, 把 SASS 文件编译成 .css 文件.

  • SASS 的 .scss 格式兼容 CSS 语法.
  • chrome 通过 sourceMap, 支持调试 sass 文件.

Install

  1. install:
    1. install ruby
    2. gem install sass
  2. command:
    1. sass --watch index.scss:output.css
    2. sass --watch app/sass:public/css
    3. --style nested/compact/expanded/compressed

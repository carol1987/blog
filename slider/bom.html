Cookie

存储到特定域名(domain)的特定目录(path)下, 存在过期时间(expires), 浏览器对 cookie 总大小有限制.

网页中请求子文件(css/js/image等)时, 请求的 head 信息中会加上 cookie - 静态资源放到其他域名下, 减少头信息数据.

Homework

  • cookie.get( key )
  • cookie.set( key, value, expires, domain, path );
  • cookie.remove( key )

参考: cookie.js

LocalStorage

localStorage 会把数据存储在本地, 并且不存在过期时间;

各个浏览器对存储空间有大小要求, 一般可存储 5M; 超出 size 限制时, 会抛出异常.

用户可手动删除 localStorage 内容.

IE 低版本不支持 localStorage, 但支持 userdata

Animation

JavaScript/UI 和 CSS 动画不在一个进程, 即: 相互不阻塞.

查看开发者工具中 animation 相关事件.

CSS

animation-delay
设置延时, 即从元素加载完成之后到动画序列开始执行的这段时间.
animation-direction
设置动画在每次运行完后是反向运行还是重新回到开始位置重复运行.(alternate)
animation-duration
设置动画一周期的时长
animation-iteration-count
设置动画重复次数; infinite 为无限次重复动画
animation-name
@keyframe 指定的关键帧名称
animation-play-state
允许停止和恢复动画. [running|paused]
animation-timing-function
设置动画速度函数 - 动画在关键帧之间是如何变化. [ease]
animation-fill-mode
动画结束时, 元素应用的样式.[node|forwards-动画结束时状态|backwords-动画结束时状态|both]

animation: name duration function delay count direction fill-mode play-state;

JavaScript - Animation Event

  • animationStart - webkitAnimationStart
  • animationIteration - webkitAnimationIteration
  • animationEnd - webkitAnimationEnd
  • event.animationName - @keyframe 名称
  • event.elapsedTime - 动画开始与现在的时间间隔

属性选择器样式演示

测试属性选择器：

当前元素为： p[foo]

当前元素为： p[foo=bar]

当前元素为: p[foo="en bar cn"]

当前元素：p[foo=bartest]

当前元素： p[foo=testbar]

当前元素： p[abbarc]

当前元素： p[foo=bar-ab]

当前元素： p[foo="ab bar-ab"]

[foo],              /* 具有 foo 属性的元素 */
[foo="bar"],        /* 具有 foo 属性, 并且属性值为 bar */
[foo~="bar"],       /* 具有 foo 属性, 并且属性使用空格分隔后, 其中有值等于 bar */
[foo^="bar"],       /* 具有 foo 属性, 并且以 start 字符开头 */
[foo$="bar"],       /* 具有 foo 属性, 并且以 end 字符结束 */
[foo*="bar"],       /* 具有 foo 属性, 并且其值包含 contain 字符 */
[foo|="bar"]        /* 具有 foo 属性, 并且以 - 线分隔, 并且 en 开头 */

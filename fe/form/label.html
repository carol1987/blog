Label 标签相关测试内容

Label 和 控件关联时, 表示为: 当点击 label 时, 焦点聚焦于控件上.

Label(L) 标签包含 input(I) 标签, 并且 label 无 for 属性, 那么 L 和 I 关联.

Label(L) 标签包含多个 input(A, B, C...) 控件, 并且 label(L) 无 for 属性, 此时, Lable(L) 和 第一个 input(A) 关联.

Label(L) 标签包含 input(I) 标签, 但是 label(L) 存在 for 属性指定到其他 control/不存在的 control, 那么 Label(L) 不与 input(I) 关联.

Label(L) 标签不包含 input(I) 标签, 但 Label(L) 使用 for 属性关联 input(I), 那么 Label(L) 和 input(I) 关联.

Label(L) 标签包含 input(I) 标签, 并且同时有其他 label(L2) 标签使用 for 属性关联 input(I), 那么 Label(L) 和 label(L2) 同时与 input(I) 关联.

多个 Label(L1, L2, ...) 标签通过 for 属性和 input(I) 关联, 那么 Label(L1, L2,....) 与 input(I) 关联.

| |

form 中书写 input(I) 控件, 而 label(L) 放在 form 外, 但 label(L) 通过 for 属性与 input(I) 的 id 关联, 那么 Label(L) 与 input(I) 关联.

  • label.htmlFor 返回 html 文件中, label 标签 for 属性的值.
  • label.control 返回与 label 元素关联的控件. [label 只能关联一个控件]
  • control.labels 返回与控件相关联的 labels. [控件可与多个 lable 关联]

Auto Transition


HTML:
<div id="demo" class="box"></div>

CSS:
.box {
  width: 100px;
  height: 100px;
  background-color: #faa;
}
.enable {
  transition: all 0.8s linear;
  transform: translate( 300px, 0 );
}

JavaScript:
var demo = document.getElementById( 'demo' );
demo.offsetWidth;
demo.classList.add( 'enable' );

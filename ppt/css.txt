BFC - Block Formatting context
块级格式化上下文, 有自己独立的布局环境和规则, 它不受外部环境影响, 也不会影响外部环境, 但是它还在普通的文档流中.

触发条件
* float: 除 none 以外的值
* overflow: 除了 visible 以外的值( hidden/auto/scroll )
* display: table-cell(table), table-caption, inline-block
* position: absolute, fixed

  zoom: 1;

OOCSS
* 抽象模块 - title/list/button/media/popup/grid/form/table/link/icon 等
* 分离容器和内容
* 结构和皮肤分离 - 皮肤(颜色/边框/图像)
* 分离轮廓和背景
* 组合大于继承
* 自适应 - 可任意移植到其他模块(位置)

SASS
Grunt/Glup

需求:
* 有一个容器, 左右两列组成
* 左侧是图片, 右侧是内容(有可能很复杂)
* 左右两侧存在间隙 - ex: 10px;
* 左侧图片的尺寸不固定( width/height 都不固定 )
* 右侧内容不能环绕左侧图片
* 右侧要求占满余下空间
* 右侧内容可能超出容器 - ex: New! 小图标
